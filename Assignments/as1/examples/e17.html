iFrames



iFrames display web pages within other web pages.




			<iframe src="URL"></iframe>


The URL is the location of the iFrame page.

You can set attributes such as height, width and even remove the border with frameborder.




			<iframe src="./e16.html" width="200" height="200">
			</iframe>
