JavaScript IDEs



IDEs (Integrated Development Environments) provide a uniform program within which you can code, edit, test and debug applications written in a given language. In this case, applications written in JavaScript. As JavaScript often goes hand in hand with other web applications such, IDEs that support other languages such as HTML, CSS, PHP, SQL, etc., are of particular note.

One IDE in this mould is eclipse, which features a highly customizable core base depending on the languages the user is interested in, conveniences such as auto-complete, code checking, code formatting and syntax checking. It also offers code previewing and can be run on a number of different platforms.

Netbeans is another IDE that follows in a similar vein. While it hangs its hat on java (much like eclipse), it similarly offers deep support for web applications that use JavaScript, including syntax highlighting, code formatting, code completion, testing support and multiple browser support. It also provides an integrated JavaScript debugger. Like eclipse, it also works on most machines.

Finally, sublime is a personal favorite. While not quite a full IDE, it is however a powerful editor with great extensibility for a large array of programming languages, including JavaScript. One of its calling cards is the vast number of keyboard commands that allow the user to quickly select desired text patterns to edit. If you're looking for something a little more lightweight, yet still very powerful, sublime is a great choice.
