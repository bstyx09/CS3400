Color Values


Color values can be used to change the color of different elements on your web page more precisely than the use of color names alone. Instead of using one of the 140 predefined color names, you can use hex values as color codes to creates millions of different colors. For example, you can have text that is white or purple. For a more comprehensive list of hex values, see the link at the bottom of the page.



<p><font color="#800080">This is some text!</font></p>
<p><font color="ffffff">This is some text!</font></p>
