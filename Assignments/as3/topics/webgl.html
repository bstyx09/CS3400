WebGL



WebGL is a standard for a 3D graphics API. It is based on OpenGL ES 2.0 and operates through the HTML5 <canvas>, much like Kinetic.JS. WebGL is Shader-based and low-level, so it follows that it can be used for some pretty complex graphics creation.

Some prerequisite areas of knowledge that are recommended for WebGL include the shading language GLSL, matrix computation for transformations and vertex operations.

Some advantages of WebGL:

  • Based on familiar and widely accepted 3D graphics standard (OpenGL)
  • Cross-browser/platform compatibility, including IE 11
  • Well integrated with HTML content
    • Hardware-accelerated 3D graphics within browsers


Khronos, the development group for WebGl, have a nice wiki as a resource for developers interested in using WebGL.
