Entities


Entities are used to display HTML reserved characters on your screen. For example, the < and > characters are displayed by writing &lt; and &gt;. Below is a table that shows some commonly used entities, along with some example code.


Result Description Entity Name Entity Number
  non-breaking space &nbsp; &#160;
< less than &lt; &#60;
> greater than &gt; &#62;
& ampersand &amp; &#38;
¢ cent &cent; &#162;
£ pound &pound; &#163;
¥ yen &yen; &#165;
€ euro &euro; &#8364;
© copyright &copy; &#169;
® registered trademark &reg; &#174; 


&lt;p&gt;A paragraph.&lt;/p&gt;
