What is CSS?


- CSS stands for Cascading Style Sheets
- Styles are used to define how to display HTML elements
- Styles were originally added to HTML 4.0
- Using external style sheets can help save a lot of work
- These external style sheets are saved as CSS files



How does it help?


CSS helps to simplify HTML documents by defining how the HTML contents are displayed.
CSS allows for the removal of most formatting tags from an HTML document and is supported by every browser today



CSS Syntax


Creating a CSS rule set requires a selector and declaration block

The selector points to the HTML element to be styled

The declaration block contains the elements to be styled

Declartions found in the declaration block contain a property name and value separated by a colon. Each declaration in the block is separated by a semicolon.
