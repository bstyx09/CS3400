Image Opacity


The opacity property is a CSS3 designation and, as the name suggests, can be used to change the opacity (or transparency) of an image.


img { opacity: 0.4; }


Image Sprites


Image sprites are collections of images that are put into a single image. One main reason for using image sprites is to reduce the number of server requests for separate images.
