What is CSS?


- CSS stands for Cascading Style Sheets
- Styles are used to define how to display HTML elements
- Styles were originally added to HTML 4.0
- Using external style sheets can help save a lot of work
- These external style sheets are saved as CSS files



How does it help?


CSS helps to simplify HTML documents by defining how the HTML contents are displayed.
CSS allows for the removal of most formatting tags from an HTML document and is supported by every browser today



CSS Syntax


Creating a CSS rule set requires a selector and declaration block



Image above pulled from w3schools.com



The selector points to the HTML element to be styled

The declaration block contains the elements to be styled

Declartions found in the declaration block contain a property name and value separated by a colon. Each declaration in the block is separated by a semicolon.



	<p {color:blue;text-align:center;}>


CSS can also contain comments. CSS comments are started with a <* and then end with *>

Here is an example:


	<* This is a comment
	Comments can be spread across
	multiple lines too. *>


CSS Selectors


CSS Selectors are used to select and modify elements in an HTML document. CSS Selectors modify HTML elements based on their attributs, id, class, type, values of attributes, etc.


The following example would center align all <h2> elements in a page and color their text blue.



	h2 {
		text-align: center;
		color: blue;
	}


CSS also allows for the selection of only specific elements to be affected by a class. using the selector p.center will only affect all p elements with class="center".
Note: Class names should never be started with a number



Grouping Selectors


When it is desired to modify more than one element with the same style, they can be consolidated into one rule with each selector separated by a comma



	h1, h2, p {
		text-align: center:
		color: blue;
	}


How to Insert a CSS


There are three ways to insert a style sheet

These three ways consist of:
an external style sheet
an internal style sheet
and inline style


External Style Sheet



An external style sheet is useful when the style should be applied to several different pages. External style sheets can change the look of a whole website with just one file.


Every page that uses the style sheet should link back to the style sheet with the <link> tag. This tag goes in the head section of the HTML document.


	<head>
	<link rel="stylesheet" type="text/css" href="stylesheet.css">
	</head>

Note: All CSS files must be named with the .css extension.




Internal Style Sheet



An internal style sheet is useful when there is only one file that should use the unique style. Internal styles are defined in the head section of an HTML document inside the <style> tag


Here is an example:



	<head>
	<style>
	body {
		background-color: linen;
	}
	h1 {
		color: blue;
		margin-left: 40px;
	}
	</style>
	</head>


Inline Styles



An inline style loses many of the advantages of the style sheet and should be used sparingly.

To use inline styles, add the style attribute to the relevant tag. The style attribute can contain any CSS property.
Here is an example:


	<h2 style="color:blue;margin-left:25px;">This is the heading.</h2>
