Tables



Table are defined with the <table> tag. Tables are divided into table rows and table data with the <tr> and <td> tags, respectively. A table row can also be divided into table headings with the <th> tag.




		<table style="width:100%">
			<tr>
				<th>City</th>
				<th>Team</th>
				<th>Wins</th>
			</tr>
			<tr>
				<td>Buffalo</td>
				<td>Sabres</td>
				<td>73</td>
			</tr>
			<tr>
				<td>Boston</td>
				<td>Bruins</td>
				<td>11</td>
			</tr>
			<tr>
				<td>Toronto</td>
				<td>Maple Leafs</td>
				<td>7</td>
			</tr>
		</table>


City Team Wins
Buffalo Sabres 73
Boston Bruins 11
Toronto Maple Leafs 7


Table elements can also be styled with CSS.




		table, th, td {
			border: 1px solid black;
		}
