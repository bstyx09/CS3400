Images



Images are defined with the <img> tag. It only contains attributes and does not have a closing tag. The src attribute defines the URL of the image. Images can be a number of different file types, such as .JPG, .GIF or .PNG.

Additionally, alternate text for an image (if it can't be displayed) may be defined using the alt attribute.




	<img src="../images/redapple.jpg" alt="a red apple">
	<img src="../images/redapple.png" alt="a red apple">
	<img src="../images/redapple.gif" alt="a red apple">




The style attribute may be used within the <img> element to set the height and width of the displayed image.




<img src="../images/redapple.jpg" alt="a red apple" style="width:256px;height:256px">





Finally, a common convention is using an image as a link instead of, or in addition, to text.




	<a href="e12.html">
<img src="../images/next.png" alt="Tables" style="width:128px;height:128px">
	</a>
