Forms



HTML forms are used to pass data to a server. They can contain input elements like text fields, checkboxes, radio-buttons, submit buttons and others.

The <form> tag is used to create an HTML form:




		<form>
			...
			input elements
			...
		</form>


The <input> element is used to select user information, and can vary in many ways, depending on the type attribute. Some of the most common input types follow.



Text Fields



The <input type="text"> attribute defines a one-line input field that a user can enter text into:




		<form>
			Name: <input type="text" name="name"><br>
			Date of Birth: <input type="text" name="dob">
		</form>

Password Field



The <input type="password"> defines a password field, and is similar to a text field implementation:




		<form>
			Password: <input type="password" name="pwd">
		</form>


Password:


If you'll notice above, characters entered into a password field are masked.



Radio Buttons



<input type="radio"> defines a radio button. Radio buttons allow a user to select only one of the choices:




	<form>
		<input type="radio" name="pet" value="dog">Dogs<br>
		<input type="radio" name="pet" value="cat">Cats
	</form>


Dogs
Cats


Checkboxes



<input type="checkbox"> defines a checkbox. Unlike radio buttons, checkboxes allow a user to select any number of options simultaneously:




<form>
	<input type="checkbox" name="education" value="highschool">Highschool Diploma<br>
	<input type="checkbox" name="education" value="somecollege">Some College<br>
	<input type="checkbox" name="education" value="college">College Diploma
</form>


Highschool
Some College
College


Submit Button



<input type="submit"> defines a submit button. The function of the submit button is to send form data to a server. The data is sent to the specified page or file in the form's action attribute. The file typically does something with the received input.




		<form name="input" action="./e17.html" method="get">
			Username: <input type="text" name="user">
			<input type="submit" value="Submit">
		</form>
