Drag and Drop



Drag and drop the Velociraptor into one of the two boxes below!





















The code follows for the HTML elements:


<img id="raptor" ondragstart="drag(event)" src="../images/Velociraptor.png">

<div id="raptor2" ondrop="drop(event)" ondragover="allowDrop(event)"></div>

<div id="raptor2" ondrop="drop(event)" ondragover="allowDrop(event)"></div>


And the code for the Javascript itself:


function allowDrop(ev) {
     ev.preventDefault();
}

function drag(ev) {
     ev.dataTransfer.setData("text/html", ev.target.id);
}

function drop(ev) {
     ev.preventDefault();
     var data = ev.dataTransfer.getData("text/html");
     ev.target.appendChild(document.getElementById(data));
}
