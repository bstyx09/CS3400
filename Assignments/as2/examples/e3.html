Layout Elements



Box Model


In HTML, elements, particularly layout elements styled with CSS, can be looked at as boxes. Hence, the term 'box model.' The CSS box model wraps around HTML elements, and consists of margins, borders, padding and the nested content. This design gives us the flexibility to define space and borders between and around elements, as illustrated below:



  • Content - where the displayed content exists, typically text and images
  • Padding - a transparent area
  • Border - a potentially visible border
  • Margin - a transparent area outside the border



Width and Height


Knowledge of the box model is crucial when designing the height and width of a layout, and the elements contained therein. It is important to note that the height and width properties of an element only affect the content area. In order to properly reflect your design, the size of the padding, borders and margin must be taken into account.



Example:


div { width: 900px; padding: 10px; border: 5px solid black; margin: 10px; }

If you just look at the width property, you might think the div element is 900px wide. But you must take into account the other elements of the box model as well. Namely, the padding, border and margin. If we add those to the content area, we see div is actually 925px wide.



Border


The border-style property specifies what kind of border to display. This property is required for any other border properties to have any affect.



Examples:


{ border-style: dotted; }

{ border-style: solid; }

{ border-style: ridge; }

{ border-style: double; }

There are a number of other border styles as well.



Border Width


The border-width property sets the width of the border. This value can be set in pixels, or in pre-defined values of thin, medium or thick. As we noted above, this property must be used in conjunction with the border-style property.



Examples:


{ border-style: solid; border-width: 5px; }

{ border-style: solid; border-width: thin; }

{ border-style: solid; border-width: medium; }

{ border-style: solid; border-width: thick; }


There are other properties related to borders as well, such as border-color. Borders can also be styled per side, with specifications such as border-top-style, etc.



Outlines


An outline is a line that is drawn around elements, including outside the border, to make the element pop. The style, color and width of an outline may be specified in this way.





Note that the outline property is not part of an element's dimensions.



Margin


The margin clears an area around an element and outside its border. The margin, as mentioned in the box model description, is completely transparent. Note that it is possible to use negative values (default units are pixels) in order to overlap content.



Padding


The padding clears an area around the content, but inside the border, of an element. The padding is affected by the background color of the element.
