Native Application Web Apps


Often times, it is useful to be able to simulate the look and feel of a native application from a web application. Luckily, both Android and iOS provide simple functions to make this possible.


Android

Android provides a simple way to create the native look and feel of an application with a web app. This is done through the use of WebViews.


First, make sure the application has access to the internet. Add the following to the application's manifest file:



<uses-permission android:name="android.permission.INTERNET"/>

Next, add a WebView element to the application's UI xml file, such as:



<WebView  
xmlns:android="http://schemas.android.com/apk/res/android"
android:id="@+id/webview"
android:layout_width="fill_parent"
android:layout_height="fill_parent"
/>

Finally, create a handle to the WebView and call the loadUrl method in your application's main java file:



WebView browser = (WebView) findViewById(R.id.webview);
browser.loadUrl("http://www.example.com");

iOS

iOS also provides a simple way to create the native look and feel of an application with a web app. This is again done through the use of WebViews.
