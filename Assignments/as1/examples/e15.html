Layout



HTML5 defines different parts of a web page with new semantic elements.



<header>
<nav>
<section> <aside>
<article>
<footer>


The <header> defines the header for a document or a section. The <nav> defines a container for navigation links. The <section> defines a section in a document. The <article> defines an independent self-contained article. The <aside> defines content aside from the content, such as a sidebar. The <footer> defines a footer for a document or section.



<div>



The <div> element is a block level element, which, as we've seen, was designed to group HTML elements. However, since CSS can position and style <div> elements, layouts can be designed using them.

The following is an abbreviated excerpt from the code that deals with the layout of this page, featuring <div> elements.




<body>

   <!-- Begin Wrapper -->
   <div id="wrapper">
   
         <!-- Begin Header -->
         <div id="header">
		 
		       ...content...
			   
		 </div>
		 <!-- End Header -->
		 
		 <!-- Begin Left Column -->
		 <div id="leftcolumn">
		 
			   ...content...

		 </div>
		 <!-- End Left Column -->
		 
		 <!-- Begin Right Column -->
		 <div id="rightcolumn">
		 
			...content...
		 
		 </div>
		 <!-- End Right Column -->
		 
		 <!-- Begin Footer -->
		 <div id="footer">
		       
			  ...content...
			    
	     </div>
		 <!-- End Footer -->
		 
   </div>
   <!-- End Wrapper -->
   
</body>
