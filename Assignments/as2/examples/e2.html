Backgrounds


CSS allows the changing of backgrounds. Users can select colors or images as a background.



h2 {
    background-color: #6495ed;
   }

Resource.


Text


CSS text allows the changing of text colors, text decoration, and formatting.



p.text {
    text-transform: lowercase;
	color: rgb(0,0,255);
}

Resource.


Font


CSS font allows the changing of texts font, style, and size.



p.serif {
    font-family: "Times New Roman", Times, serif;
	font-style: italic;
	font-size: 14px;
}

Resource.


Links


CSS allows one to change the style of links throughout the page.



a:link {
    color: #0033CC;
}

a:visited {
    color: #3399FF;
}

a:hover {
    color: #FF00FF;
}

a:active {
    color: #0000FF;
}

Resource.


Lists


CSS allows one to change the style of lists throughout the page.


Example of unordered lists:

  • Coffee
  • Tea
  • Coca Cola

Example of ordered lists:

  1. Coffee
  2. Tea
  3. Coca Cola


ul.a {
    list-style-type: circle;
}

ol.d {
    list-style-type: lower-alpha;
}
