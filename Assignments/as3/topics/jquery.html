jQuery



What is jQuery? From jQuery itself:

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.


In general, the purpose is to make it easier to use JavaScript by wrapping common tasks in callable methods that reduce code typed and time spent. The jQuery library contains the following features:



  • HTML/DOM manipulation
  • CSS manipulation
  • HTML event methods
  • Effects and animations
  • AJAX (Asynchronous JavaScript and XML)
  • Utilities

Furthermore, with its popularity, jQuery runs exactly the same in all major browsers, as the developers have internally accounted for cross-browser compatibility.
