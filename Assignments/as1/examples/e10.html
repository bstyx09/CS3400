CSS


CSS (cascading style sheets) outline how individual HTML elements should be displayed, or styled. This can be achieved through 3 different methods.

The first is called inline styling. This method uses the style attribute in HTML elements.

For example:




<p style="color:red; margin-left:20px">This is a red paragraph with a left margin.</p>


The second is called internal styling, or using the <style> element in the <head> section.

For example:




<head>

<style>
	body {background-color:gray}
	p	 {margin:10px}
</style>

</head>


Finally, the third method uses external style sheets, which are defined within the <link> element in the <head> section.

For example, part of the external style sheet for this site is declared as follows:




#leftcolumn {
 font-size: medium;
 font-weight: bold; 
 color: #333;
 border: 1px solid #ccc;
 background: #C7C7F9;
 margin: 0px 5px 5px 0px;
 padding: 10px;
 height: auto;
 width: 225px;
 float: left;
}
#rightcolumn { 
 float: right;
 color: #333;
 border: 1px solid #ccc;
 background: #E7E8F3;
 margin: 0px 0px 5px 0px;
 padding: 10px;
 height: auto;
 width: 648px;
 display: inline;
}


As you can see, CSS can be used in different ways to specify a wide variety of HTML elements.
