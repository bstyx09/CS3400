Lists



Lists in HTML consist of Unordered Lists, Ordered Lists and Description Lists. These lists can be styled with CSS.



Unordered



Unordered lists can by styled to define the marker style. 


			   <ul style="list-style-type:square">
				<li>Vacuum</li>
				<li>Dust</li>
				<li>Dishes</li>
			   </ul>


  • Vacuum
  • Dust
  • Dishes



			   <ul style="list-style-type:disc">
				<li>Vacuum</li>
				<li>Dust</li>
				<li>Dishes</li>
			   </ul>


  • Vacuum
  • Dust
  • Dishes



			   <ul style="list-style-type:circle">
				<li>Vacuum</li>
				<li>Dust</li>
				<li>Dishes</li>
			   </ul>


  • Vacuum
  • Dust
  • Dishes


Ordered



Ordered lists can include a type attribute to define the marker type.




			   <ol type="1">
				<li>Vacuum</li>
				<li>Dust</li>
				<li>Dishes</li>
			   </ol>


  1. Vacuum
  2. Dust
  3. Dishes



			   <ol type="A">
				<li>Vacuum</li>
				<li>Dust</li>
				<li>Dishes</li>
			   </ol>


  1. Vacuum
  2. Dust
  3. Dishes



			   <ol type="a">
				<li>Vacuum</li>
				<li>Dust</li>
				<li>Dishes</li>
			   </ol>


  1. Vacuum
  2. Dust
  3. Dishes



			   <ol type="I">
				<li>Vacuum</li>
				<li>Dust</li>
				<li>Dishes</li>
			   </ol>


  1. Vacuum
  2. Dust
  3. Dishes



			   <ol type="i">
				<li>Vacuum</li>
				<li>Dust</li>
				<li>Dishes</li>
			   </ol>


  1. Vacuum
  2. Dust
  3. Dishes


Description



A description list contains a description of each term. The <dt> tag defines the term, and the <dd> tag defines the data.




				<dl>
					<dt>22 / 7</dt>
					<dd>... > pi</dd>
					<dt>Fibonacci Sequence</dt>
					<dd>... - 1 + 2 + 3 + 5 + 8 + ...</dd>
				</dl>
