Standard Locations


Internal


In HTML content, JavaScripts must be inserted between <script></script> tags. They can be put in both the <body> and the <head> section of an HTML page. The <script></script> tags indicate where the JavaScript starts and ends.


<script>
     function myFunction() {
          document.getElementById("demo").innerHTML = "My First JavaScript Function";
     }
</script>


External


JavaScripts can also be sourced externally, which may be preferable when a particular JavaScript is used more than once, or simply for good separation design. JavaScript files have the .js extension. In order to reference an external script from within HTML, place the name of the script in the src attribute of the <script> tags.


<script src="script.js"></script>
