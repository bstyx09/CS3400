Standards and Best Practices


Avoid Global Variables


Global variables and functions can be overwritten by other scripts, so generally local variables with closures are preferred. To that end, always declare local variables with the var keyword.


Primitives vs. Objects


Numbers, strings and booleans are primitive values in JavaScript. As such, they should not be declared as objects. Doing so is inefficient and can lead to unintended side effects.


     var x = "hello";     // Correct primitive value declaration
     var y = new String("hello");     // Incorrect primitive declaration as object

Furthermore, when declaring primitive variables and objects, don't use the new identifier. Instead, follow the following list:


  • Use {} instead of new Object()
  • Use "" instead of new String()
  • Use 0 instead of new Number()
  • Use false instead of new Boolean()
  • Use [] instead of new Array()
  • Use /(:)/ instead of new RegExp()
  • Use function (){} instead of new function()

     var x1 = {};      // new object
     var x2 = "";      // new primitive string
     var x3 = 0;      // new primitive number
     var x4 = false;      // new primitive boolean
     var x5 = [];      // new array object
     var x6 = /()/;      // new regexp object
     var x7 = function(){};      // new function object

Finally, keep in mind that JavaScript is loosely typed, and so variables can be assigned different data types and be correct within the lexical framework of JavaScript.


     var x = "hello";      // typeof x is a string
     x = 5;      // changes typeof x to a number

Use === Comparison


The === operator will force the comparison of values and type, whereas the preferably avoided == operator will always convert the expression to uniform data types before the comparison.


Parameter Defaults


Calling a function without all of the defined arguments sets the missing arguments to undefined, which can do bad things within the function. This behavior can be caught in some error checking which assigns default values to arguments:


     function myFunction(x, y) {
           if (y === undefined) {
                y = 0;
           }
     }
