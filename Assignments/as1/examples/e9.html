Head


The <head> element is a container that is placed above the <body> tags and is not displayed within the page directly. It typically includes instructions to find css style sheets and meta information.

The following tags can be included in the head sections: <title>, <style>, <meta>, <link>, <script>, <noscript> and <base>.

For example, the <head> element of this page is as follows: 


	<head>
	<meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1" />
	<title>CS 3400</title>
	<link rel="stylesheet" type="text/css" href="../main.css" />
	</head>


<title>


The <title> tag defines the title of the document, and is required in all HTML/XHTML documents. Title definition includes the display in the browser toolbar, the favorites menu and in search-engine results.



<base>


The <base> tag sets the base URL for all relative URLs in a page.



<link>


The <link> tag defines the relationship between a document and an external source, which is most often a style sheet.



<style>


The <style> tag defines style information for an HTML document. This information specifies how browsers should render HTML elements.

This tag is often separated into its own style sheet, which is then referenced by the <link> tag as noted above.



<meta>


The <meta> tag outlines metadata, or information about data, within the HTML document. HTML metadata typically consists of page descriptions, keywords, last modified and more.



<script>


The <script> tag defines a client-side script, ie, JavaScript.
