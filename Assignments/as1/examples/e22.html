Symbols


Many mathematical, technical, and currecy symbols, are not presented on a normal keyboard. To add these symbols to an HTML page, you can use an HTML entity name. If no entity name exists, you can use an entity number; a decimal (or hexadecimal) reference. For example, the euro, €
